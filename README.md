## DE-Project---Data-processing-with-Kafka

**Mô tả:** Project này xây dựng ứng dụng Machine Translation. Tiếp đó, xây dựng một data pipeline để chuyển dữ liệu (giả lập) hành vi người dùng trên ứng dụng lên Kafka. Sau đó, sử dụng kho dữ liệu StarRocks để lưu trữ dữ liệu được kéo về từ Kafka. Cuối cùng, sử dụng Apache Superset để trực quan hóa dữ liệu đã lưu trữ tại StarRocks

**Công nghệ sử dụng:** Python, Apache Kafka, Apache Superset, StarRocks
